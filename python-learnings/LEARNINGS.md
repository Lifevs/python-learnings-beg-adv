## 🟢 Day 1: Python Foundations & Type System
*Focus: Getting started, basic syntax, and how Python handles data.*

### 1. Variables & Syntax Basics
- **Variable Declaration**: Declared simply as `name = value`. No `let`, `var`, or `const` keywords are required.
- **Naming Rules**:
  - Must start with a letter or underscore (`_`).
  - Case-sensitivity is key: `user_age` is different from `User_Age`.
  - Avoid reserved keywords (like `if`, `else`, `print`).
  - **Best Practice**: Use descriptive names rather than single letters (e.g., `account_balance` vs `a`).

### 2. Output & Basic Types
- **The `print()` Function**: Separating values with commas (e.g., `print(val1, val2)`) automatically inserts a space.
- **Dynamic Typing**: Python is dynamically typed—it automatically detects if a value is a string, number, or boolean based on the assignment.
- **Type Inspection Tools**:
  - `type()`: Identifies the class (e.g., `<class 'int'>`).
  - `isinstance()`: Checks if a variable belongs to a specific type or a group of types.

## 🟡 Day 2: String Mastery & Manipulation
*Focus: Deep dive into text processing and formatting.*

### 1. String Properties
- **Immutability**: Once created, strings cannot be modified in place. You must create a new string to change a character.
- **Length**: Measured using the built-in `len()` function.
- **Multiline Text**: Defined using triple quotes (`"""` or `'''`), allowing text to span multiple lines naturally.

### 2. Formatting & Methods
- **Concatenation**: Joining strings together using the `+` operator.
- **Interpolation**: The process of embedding variables and expressions directly into strings for dynamic content.
- **Essential Methods**:
  - `upper()` / `lower()`: Change character casing.
  - `isupper()` / `islower()`: Boolean checks for casing.
  - `replace(old, new)`: Swaps specific parts of a string.
  - `str()`: Safely converts any object into a string format for display or joining.

## 🔴 Day 3: Numbers, Operators & Career Application
*Focus: Mathematical logic, efficiency, and real-world goals.*

### 1. Numeric Logic & Operators
- **Integers vs. Floats**: Integers are whole numbers; Floats handle decimals.
- **Floor Division (`//`)**: A specialized operator that divides and rounds down to the nearest whole number, discarding the remainder.
- **Absolute Values**: Handled via the `abs()` function.
- **Augmented Assignment**: Using shortcuts like `+=`, `-=`, and `**=` to update variables concisely.
- **Syntax Warning**: Python does **not** support `++` or `--` increment/decrement operators. Use `x += 1` instead.
